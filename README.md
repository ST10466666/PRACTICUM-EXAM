# PRACTICUM-EXAM
Music Playlist Manager 
## Features
Add up to 4 songs with details:
  - Song title
  - Artist name
  - Rating (1-5 stars)
  - Optional comments
- View complete playlist
- Calculate average song rating
- Clean exit with confirmation
